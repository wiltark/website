Register a font.

## Usage

```
$registerFont[src;name?]
```

## Parameters

| Field | Description           | Type   | Required |
| ----- | --------------------- | ------ | -------- |
| src   | The font source.      | string | true     |
| name  | The font family name. | string | false    |
