title: $createCanvas

---

Creates a new canvas.

## Usage

```
$createCanvas[canvas;width?;height?]
```

## Parameters

| Field  | Description        | Type   | Required |
| ------ | ------------------ | ------ | -------- |
| canvas | The canvas name.   | string | true     |
| width  | The canvas width.  | number | false    |
| height | The canvas height. | number | false    |
